Минимальная канбан-доска

Эта версия работает без установки.

Как запустить:
1. Дважды щёлкните по файлу Start Kanban.exe.
2. Дождитесь открытия окна программы.
3. Никакие терминалы, npm, python, node или другие ручные команды не нужны.

Что важно:
- локальный API стартует автоматически
- данные сохраняются между перезапусками
- закрытие окна корректно останавливает внутренние компоненты
